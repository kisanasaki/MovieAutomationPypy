# 概要
これは動画編集の一部を自動化するツールです。具体的には、全体の工程（1～8）のうち3～5を自動化することを目的としています。

1. 企画
    -  動画のテーマを決める
    -  動画の構成を決める
2. 撮影
3. 編集
    -  ジャンプカット 
    -  テロップ
    -  効果音自動挿入　←ｲﾏｺｺ
4. 動画の確認
